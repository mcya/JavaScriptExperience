## for循环 ##


固定次数： 起始值， 终止值， 增量 
            1      100    1   ==> 循环100次
            1      100    2   ==> 循环50次
            1      5      1   ==> 循环5次


语法：
for (var i = 起始值; i <= 终止值; i++ ) {
    循环的内容
}

## do-while循环 ##

// 先执行循环体， 再判断条件
do {
	document.write(i + ", ");
	i++;
} while (i <= 5);


do-while 不管条件如何，总会执行第一次。

语法：

do {
    循环语句(if, while,for, do-while)
} while( 表达式 );
